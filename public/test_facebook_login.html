‚ùå Troubleshooting

Jika kembali ke halaman login tanpa error:

  1. Buka browser DevTools (F12) ‚Üí Tab Network
  2. Klik tombol "Test Facebook Login" di atas
  3. Lihat request yang terjadi:
    • Harus ada redirect ke facebook.com
    • Setelah login di Facebook, harus ada callback ke /auth/facebook/callback
  4. Jika langsung kembali tanpa ke Facebook, cek log error di storage/logs/laravel.log

Error umum:

  • "Can't Load URL" ‚Üí Periksa App Domains dan Valid OAuth Redirect URIs
  • "Invalid App ID" ‚Üí Periksa FACEBOOK_CLIENT_ID di .env
  • "InvalidStateException" ‚Üí Clear browser cookies dan cache

Untuk troubleshooting lengkap, baca file FACEBOOK_LOGIN_TROUBLESHOOTING.md

üìä Check Laravel Log

Atau di PowerShell:
